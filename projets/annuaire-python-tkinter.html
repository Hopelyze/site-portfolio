2020 - Annuaire numérique en Python avec Tkinter

Lycée NSI - Projet en solo

Description

Application graphique en Python permettant de gérer un annuaire numérique. Interface conçue avec Tkinter pour offrir une interaction simple et efficace.

Fonctionnalités

  • Enregistrement et recherche de contacts
  • Utilisation d'un fichier texte comme base de données
  • Pavé numérique interactif
  • Boutons facilitant l'entrée des données

Compétences

  • Programmation Python
  • Interface graphique Tkinter
  • Gestion de fichiers (open, write, read)
  • Manipulation de widgets (Label, Entry, Button, Canvas)
  • Gestion des événements utilisateur
