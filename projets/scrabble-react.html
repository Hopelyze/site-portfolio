2025 - Scrabble en React

L2 Informatique

Description

Application web interactive permettant de jouer au célèbre jeu Scrabble.

Fonctionnalités

  • Placement de mots
  • Calcul automatique des points
  • Suggestion du mot optimal en fonction du plateau et du chevalet

Choix techniques

  • React pour mise à jour dynamique de l'interface
  • Composants réutilisables
  • Méthodologie SCRUM (approche agile)
